# История изменений

## v0.2.1
- Историю изменений теперь можно смотреть прямо в приложении: Настройки → «Обновление
  приложения» → «Что нового».

## v0.2.0

### Диагностика (по многоагентному ревью, 88 юнит-тестов)
- **Связь:** устранён «вечный рассинхрон кадров» после таймаута (дренаж буфера),
  безопасный fallback сокета, отмена чтения, валидация инициализации адаптера и
  детект мёртвого/поддельного клона, устранён краш при отказе в Bluetooth-разрешении.
- **Парсинг OBD:** объединение масок PID по всем ЭБУ, классификация отрицательного
  ответа 0x7F, фикс сборки ISO-TP длинных ответов.
- **Агент:** `clear_dtcs` проверяет реальный ответ ЭБУ; `read_live_data` различает
  «неизвестный PID» и «нет данных»; при лимите шагов агент подводит вердикт.
- **Коды:** плашка «можно ли ехать» и окраска по опасности на экране кодов,
  предупреждение об ошибке шины по режимам, подсказки по B/C/U-кодам.
- **РФ-моат:** офлайн-распознавание VIN Solaris/Rio/Creta (Z94) без 15-секундного
  зависания, связка VIN↔каталог, +18 старых китайских моделей и +3 марки (Geely MK…).
- **Устойчивость:** отмена диагностики при отключении, сохранение поездки при закрытии,
  API-ключ убран из наблюдаемого состояния.

### Новое
- **Настройки поведения:** защита аккумулятора, «не гасить экран», частота опроса приборов.
- **Автообновление** из GitHub Releases: проверка при запуске, уведомление о новой
  версии, установка одной кнопкой.
